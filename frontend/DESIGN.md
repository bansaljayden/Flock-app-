# Design System: Flock

> Single source of truth for Flock's visual language. Generated by auditing the
> live tokens in `src/index.css` + `src/App.js` (stitch-design-taste format,
> cross-checked against impeccable / high-end-visual-design / web-design-guidelines).
> Flock is a **product app** (multi-step social coordination for Gen Z), not a
> marketing site — design SERVES the product. Keep it warm, tactile, legible.

## 1. Visual Theme & Atmosphere
Warm, tactile, "soft daylight" consumer app. Density **Daily-App Balanced (4)** —
compact but breathing, never a data cockpit. Variance **Low-offset (3–4)** — a
product UI, so symmetry and predictability aid usability over artsy asymmetry.
Motion **Fluid (5)** — spring-eased micro-interactions, a live "pulse" language,
no cinematic scroll-hijacking. The feel is a cream paper surface under teal task
light: friendly, physical (film grain + soft ambient shadows), quietly premium.

## 2. Color Palette & Roles
Light mode (`:root`):
- **Cream Canvas** (`#f1ede0`) — primary app background (`--bg-primary`)
- **Pure Surface** (`#ffffff`) — cards, modals (`--bg-card-solid`)
- **Warm Tertiary** (`#e8e0d5`) — inset wells, input fills (`--bg-tertiary`)
- **Ink** (`#1e293b`) — primary text (`--text-primary`)
- **Slate Secondary** (`#556274`) — secondary text — raised for WCAG AA (`--text-secondary`)
- **Muted Tertiary** (`#78859a`) — meta/hints — raised toward AA (`--text-tertiary`)
- **Teal** (`rgb(13,148,136)` / `#0d9488`) — THE single brand accent: primary CTAs,
  active states, focus rings, ambient orbs, distance/map cues
- Semantic only: amber (warnings/moderate), red (danger/busy), green (go/available)

Dark mode (`[data-theme="dark"]`): navy-slate family (`#0f172a` canvas, `#1e293b`
surfaces), teal accent preserved, muted text lightened for AA.

**Locks & bans:**
- **One accent: teal.** No second accent. (The old indigo/purple "Birdie" orbs were
  removed — purple/indigo is the #1 AI fingerprint and clashed with teal.)
- No pure `#000000` / pure `#ffffff` backgrounds (cream + off-navy carry depth).
- Night-mode badge purple is the one sanctioned exception (semantic: moon = night).

## 3. Typography Rules
- **Family:** **Satoshi** (self-hosted, weights 400/500/700/900) on everything.
  Bundled from `src/fonts/` so it renders offline in the Capacitor WebView.
- **Hierarchy by weight, not scale:** 800 for screen titles (~20px), 700 for card
  headings, 600 for labels/emphasis, 500 for muted meta, 400 body. Tight tracking
  on titles (`-0.3px` to `-0.4px`).
- **Numbers use `tabular-nums` globally** — stats, crowd %, budgets, timestamps
  align in columns (set on `body`).
- **Banned:** Inter/Roboto/system-default as the visible face; serif anywhere
  (this is a product UI); loading fonts from a CDN (must be self-hosted/offline).

## 4. Component Stylings
- **Buttons:** the `.glass-*` system is Flock's signature — teal `.glass-primary`
  for actions, `.glass-secondary` for neutral, `.glass-danger` for destructive.
  Tactile `scale(0.98)` on `:active`, `scale(1.02)` on hover. Glass is a
  *deliberate identity choice here* (premium-consumer register), not decoration —
  keep it purposeful, don't add more blur layers.
- **Cards:** rounded 12–16px, `--bg-card-solid` with a 1px `--border-default`
  hairline AND a soft ambient shadow (`--card-shadow-sm`) so they lift off the
  cream instead of reading flat. Avoid nesting cards inside cards.
- **Elevation:** layered soft-ambient shadows (`--card-shadow` / `--card-shadow-sm`),
  tinted navy in light, deep black in dark — never a single harsh drop shadow.
- **Inputs:** label above, error below; readable fill (`--bg-tertiary`), teal focus.
- **Surfaces:** a fixed film-grain overlay (0.022) gives editorial texture.

## 5. Layout Principles
- Mobile-first, single column (this is a phone app in a WebView). `100dvh`-aware.
- Compact header + scrollable content pattern (see `HomeScreen`).
- Bottom tab nav (Nest / Discover / Messages / Profile).
- Consistent 12–16px radii; 8–16px gaps; generous-but-tight padding.

## 6. Motion & Interaction
- Spring easing `cubic-bezier(0.34,1.56,0.64,1)` for tactile pops (FAB, greeting).
- A live **"pulse"** language: gentle 2s occupancy/status pulses on active dots.
- Staggered `.scroll-fade` reveals on lists (IntersectionObserver, not scroll listeners).
- **`prefers-reduced-motion` honored** — infinite orb/shimmer/pulse loops stop,
  transitions collapse, scroll-reveal content forced visible. Non-negotiable a11y.
- Animate only `transform`/`opacity`; grain on a fixed pseudo-element only.

## 7. Anti-Patterns (Banned)
- No second accent color; no indigo/purple "AI gradient".
- No CDN-loaded fonts (offline-first); no Inter/system default as the face.
- No flat bordered cards with no elevation; no nested cards.
- No muted gray text below WCAG AA (`--text-tertiary` floor raised for this reason).
- No animation without a `prefers-reduced-motion` fallback.
- No emojis as UI icons (use the in-app `Icons.*` SVG set).
- No serif; no pure black/white surfaces; no glass added purely for decoration.
